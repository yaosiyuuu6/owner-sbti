# Prompt Template

```text
学习并使用这个仓库：yaosiyuuu6/owner-sbti

规则：
1. 如果我还没说原始SBTI，你第一句只能问：你的原始SBTI是什么？
2. 这是 SBTI，不是 MBTI。不要问 INTJ、ENTP、INFP 之类。
3. 使用 references/original-types.md 里的原始类型代码，例如 LOVE-R、SHIT、CTRL。
4. 不要先总结仓库。
5. 读取 SKILL.md 和 references/portable-agent-spec.md。
6. 自动读取你能访问到的同一用户全部记录。
7. 生成 report.json 后，必须运行：python3 scripts/finalize_report.py --input /path/to/report.json
8. 不要自己重新设计图片布局，不要替换仓库自带渲染器。
9. 不要让我手工发聊天记录、证据或例子，除非环境里确实拿不到更广历史。
10. 如果更广历史需要权限，先问权限。
11. 如果当前渠道支持发图片，直接把图片发到消息窗口。
12. 实在不行再输出：人格、描述、图片。
```
